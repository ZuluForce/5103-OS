\subsection{\-Detailed \-Description}
\-This was initially gonig to ge the abstract class for all devices such as the clock, character and block devices but it turned out this was not the desired interface for everything. \-This is why the \hyperlink{classcAbsQueuedDevice}{c\-Abs\-Queued\-Device} class was made. \-This class remains here because it is used by the clock device however it is no longer necessary.
